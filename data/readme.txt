The graph data files should be named and formed as following:

Folder: graph

Edges file: graph.txt
source_node_id target_node_id

Distances file: matrix_graph.txt
matrix form, a comma after each digit.

Communities file: graph_communities.txt
# Input: (file path)
# Nodes: (nodes number)    Edges: (edges number)
# Communities: (cummunities number)
# NId	CommunityId
node_id\tcommunit_id

Circles file: graph_circles.txt
Node indexes for a circle list in one row, a comma after each digit.
